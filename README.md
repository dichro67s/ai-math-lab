# SNU AI Mathematics Lab Website

GitHub Pages deployment package for the SNU AI Mathematics Lab website.

## Structure
- `index.html`: homepage
- `assets/`: CSS, JS, images
- `people/`: members
- `publications/`: papers and preprints
- `projects/`: research projects
- `benchmarks/`: benchmarks
- `leaderboard/`: public results
- `news/`: announcements

## GitHub Pages deployment
1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub: `Settings → Pages`.
4. Choose `Deploy from a branch`.
5. Select `main` and `/root`.
6. Save.

## Notes
- The site uses relative paths and should work directly on GitHub Pages.
- `.nojekyll` is included for safe static hosting.
